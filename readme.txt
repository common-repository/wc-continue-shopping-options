== Description ==

Most of the time as a Shop owner we always want to direct the customer at right place whenever customer is at Cart page to show related products or products with maximum profit. So with this plugin we have different options available at dashboard so we can direct the customer at right place.We have these options available: <blockquote> - Homepage<br /> - Shop<br /> - Most recently visited Product<br /> - Most recently visited Category<br /> - Your own custom link </blockquote>  == Installation ==1. Upload the folder to the `/wp-content/plugins/` directory2. Activate the plugin through the 'Plugins' menu in WordPress3. Go to WooCommerce -> Settings -> Products -> General ( Please Check : Redirect to the cart page after successful addition ).

4. Go to WooCommerce -> Settings -> Products -> WC Continue Shopping Options to change options available at cart.
